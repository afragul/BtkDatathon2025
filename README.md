# 📊 BTK Datathon 2025 – Veri Bilimi Projesi

Bu depo, **BTK Akademi tarafından düzenlenen BTK Datathon 2025** yarışması için geliştirdiğim veri bilimi çözümünü içerir. Yarışma kapsamında sunulan veri seti üzerinden analiz, modelleme ve sonuç çıkartma süreçlerini bir **Jupyter Notebook** içinde gerçekleştirdim.

---

## 📌 Proje Hedefi

Bu proje ile amaçlanan:

- Verilen veri seti üzerinde **keşifsel veri analizi (EDA)** yapmak  
- Özellik mühendisliği ve modelleme teknikleri uygulamak  
- En iyi performansı veren makine öğrenme modellerini değerlendirip seçmek  
- Yarışma formatında başarı elde etmektir

---

## 🧠 Kullanılan Teknolojiler

Bu proje aşağıdaki teknolojilerle geliştirilmiştir:

- 🐍 Python  
- 📓 Jupyter Notebook  
- 📊 Pandas / NumPy  
- 📈 Scikit-learn  
- 📉 Matplotlib / Seaborn
